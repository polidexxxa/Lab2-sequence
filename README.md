# Lab2-sequence
Realisation of a sequence in C++

## Windows MSVC

### Собрать всё
nmake /f Makefile.msvc

### Запустить
nmake /f Makefile.msvc run

nmake /f Makefile.msvc test

nmake /f Makefile.msvc demo

### Очистить
nmake /f Makefile.msvc clean


## Linux

### Собрать всё
make

### Собрать только main
make main

### Запустить main
make run

### Запустить тесты
make test

### Запустить демонстрации
make demo

### Очистить
make clean

### Пересобрать
make rebuild
