# Lab2-sequence
Sequence implementation in C++

## Windows MSVC

### Собрать всё
nmake /f Makefile.msvc

### Запустить
nmake /f Makefile.msvc run

nmake /f Makefile.msvc test

nmake /f Makefile.msvc demo

### Очистить
nmake /f Makefile.msvc clean


## Linux

### Собрать всё
make

### Запустить main
make run

### Запустить тесты
make test

### Запустить демонстрации
make demo

### Очистить
make clean
